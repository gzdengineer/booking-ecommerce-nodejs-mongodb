# Booking & E-Commerce Node.js MongoDB Project

This project is a backend application developed using Node.js, Express, and MongoDB. It provides basic booking and e-commerce functionalities such as user authentication, product management, and order processing.

---

## 🚀 Features

- User registration and login
- JWT-based authentication
- Role-based access (User / Admin)
- Product CRUD operations
- Order management system
- MongoDB database integration
- RESTful API structure

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JSON Web Token (JWT)
- dotenv
- bcrypt
